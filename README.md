# Pizza Website 

Pizza markasını tanıtmak ve menüsünü kullanıcılarla paylaşmak amacıyla geliştirilmiş basit bir web sitesi projesidir.

## Özellikler

*  Pizza markası tanıtımı
*  Menü bölümü
*  Pizza görselleri
*  Basit ve kullanıcı dostu tasarım
*  Responsive tasarım

## Kullanılan Teknolojiler

* HTML
* CSS

## Projeyi Çalıştırma

Projeyi bilgisayarınıza klonlayın:

```bash
git clone https://github.com/ALPERSERHAT5/PizzaWebsites.git
```

Proje klasörüne girin:

```bash
cd PizzaWebsites
```

Ardından `index.html` dosyasını herhangi bir web tarayıcısıyla açın.
